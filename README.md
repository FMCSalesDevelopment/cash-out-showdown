# Cash-Out Showdown! (Freedom Mortgage Training)

Interactive, brand-aligned learning game for new Call Center Loan Officers.

## Run locally
- Open `index.html` or use VS Code + “Live Server”.

## Deploy (GitHub Pages)
- Settings → Pages → Deploy from Branch → `main` `/root`.
- URL: https://<your-username>.github.io/cash-out-showdown/

## Brand & ADA
- Uses 2025 web palette (Web Blue `#1D3146`, Bright Blue `#499CC9`/`#6ED8FF`, Teal `#00838F`/`#35CCBA`, Gold `#A18732`) and **Poppins** font.  
- Visible focus outlines, high contrast, no flashing, proper `aria-` live regions and keyboard nav.

## Content & Rules
- 20 questions + bonus round; points start at 1000 and drop every 3 seconds (tiers: 1000/750/500/250/100; wrong=0).
- Shows answer, quick explanation, cumulative leaderboard.

*Internal training only; verify guidelines against current FHA, VA, and Agency Selling Guides.*
